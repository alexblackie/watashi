My Setup

I take great enjoyment of reading about how other people use their computers and desks. Sites like UsesThis and Simple Desks have taken many hours of my life, and I have taken inspiration from them many times. I realised that I don't really talk about my setup, outside of the commits for my dotfiles, so this page will serve as a perpetually-updated reference for how I'm currently using my computer.

Software

I'm currently running the latest Fedora with the default GNOME 3 desktop. I've been a GNOME guy all my life, and I really like some of the modern desktop niceties like easy configuration, 3D compositing, and a generally pleasant and cohesive UI. I stick with a lot of the standard GNOME tools: GNOME Terminal, Nautilus, Boxes, etc.

All of my development is done within Docker, using a few custom containers I've built to support all the environments I commonly use. For the basics, I rely heavily on vim, tmux, and Git. I switch every couple months between Chrome and Firefox.

Hardware

My primary machine is a Lenovo Thinkpad T460p, with an Intel i5-6440HQ CPU, 16GB RAM, 512GB SSD, and 1080p IPS display. It was generously provided by my employer.

My secondary/home computer is still my Lenovo Thinkpad X250, with an Intel i5-5200U, 8GB RAM, 512GB SSD, 20-hour average battery life, and 12.5" IPS 1366x768 display. I wrote a review of this machine previously, and still absolutely love it.

For displays, I use a single Asus PB278Q. For HIDs, I have a Filco Majestouch 2 TKL (Browns), and the glorious Logitech MX Master. However, a lot of the time I just work on the laptop itself as the 1080p screen is plenty for my needs.

For audio, I have a O2+ODAC from Mayflower Electronics, which is driving a pair of AKG K7XX Massdrop-edition headphones.
